software-fj-fase4
Sistema integral de gestión de clientes, servicios y reservas para Software FJ

Descripción
Este proyecto es un sistema desarrollado en Python utilizando Programación Orientada a Objetos (POO), el cual permite gestionar clientes, servicios y reservas para una empresa ficticia llamada Software FJ.
El sistema no utiliza bases de datos, toda la información se maneja mediante objetos, listas y archivos de texto para registro de errores.

Tecnologías utilizadas
Python Programación Orientada a Objetos (POO) Manejo de excepciones Archivos de texto (logs)

Estructura del proyecto
main.py → Programa principal interactivo cliente.py → Gestión de clientes servicio.py → Servicios disponibles (herencia y polimorfismo) reserva.py → Gestión de reservas excepciones.py → Excepciones personalizadas logs.txt → Registro de errores
¿Cómo ejecutar?
    Clonar el repositorio:
    Abrir el proyecto en visual studio code
    Ejecutar el archivo principal

Funcionalidades
Registro de clientes
Selección de servicios
Creación de reservas
Cálculo de costos según servicio
Manejo de errores controlado
Registro de errores en logs
Sistema interactivo por consola
